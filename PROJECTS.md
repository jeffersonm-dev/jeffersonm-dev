## 📡 `01` — API .NET Core + Swagger

> **Documentación interactiva y testing automatizado de APIs RESTful**

![.NET Core](https://img.shields.io/badge/.NET_Core-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)
![Swagger](https://img.shields.io/badge/Swagger-85EA2D?style=for-the-badge&logo=swagger&logoColor=black)
![JWT](https://img.shields.io/badge/JWT-000000?style=for-the-badge&logo=jsonwebtokens&logoColor=white)
![Entity Framework](https://img.shields.io/badge/EF_Core-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)

### ¿Qué hace?

API RESTful completa construida con .NET Core, documentada con Swagger UI para pruebas interactivas directamente desde el navegador. Implementa autenticación JWT segura y acceso a datos con Entity Framework Core.

### ✅ Características

- Documentación automática con Swagger / OpenAPI 3.0
- Autenticación y autorización con JWT Bearer tokens
- CRUD completo con Entity Framework Core
- Manejo de errores y respuestas estandarizadas
- Pruebas unitarias con xUnit

### 🛠 Stack

| Capa | Tecnología |
|------|-----------|
| Framework | .NET Core 7 |
| Documentación | Swagger / OpenAPI |
| Auth | JWT Bearer |
| ORM | Entity Framework Core |
| Testing | xUnit |

[![Ver código](https://img.shields.io/badge/Ver_Código-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/jeffersonm-dev/api-dotnet-swagger)

## 🖥️ `02` — App Vue.js + SQL Server

> **Aplicación full stack moderna con Vue.js 3 y backend .NET Core**

![Vue.js](https://img.shields.io/badge/Vue.js_3-4FC08D?style=for-the-badge&logo=vue.js&logoColor=white)
![Vuex](https://img.shields.io/badge/Vuex-4FC08D?style=for-the-badge&logo=vue.js&logoColor=white)
![.NET Core](https://img.shields.io/badge/.NET_Core-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)
![SQL Server](https://img.shields.io/badge/SQL_Server-CC2927?style=for-the-badge&logo=microsoft-sql-server&logoColor=white)

### ¿Qué hace?

Aplicación full stack donde el frontend Vue.js 3 consume una API RESTful construida en .NET Core. La gestión de estado se maneja con Vuex y los datos persisten en SQL Server.

### ✅ Características

- SPA con Vue.js 3 + Composition API
- Gestión de estado global con Vuex
- Consumo de API RESTful con Axios
- Diseño responsive con CSS moderno
- Base de datos SQL Server con consultas optimizadas

### 🛠 Stack

| Capa | Tecnología |
|------|-----------|
| Frontend | Vue.js 3 + Vuex |
| Backend | .NET Core |
| Base de datos | SQL Server |
| HTTP Client | Axios |
| Estilos | CSS3 / SCSS |

[![Ver código](https://img.shields.io/badge/Ver_Código-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/jeffersonm-dev/vue-sql-app)

## 🗃️ `03` — Gestor de Base de Datos

> **Sistema de gestión multi-motor con MySQL y PostgreSQL**

![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=for-the-badge&logo=mysql&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-316192?style=for-the-badge&logo=postgresql&logoColor=white)
![.NET Core](https://img.shields.io/badge/.NET_Core-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)

### ¿Qué hace?

Sistema de gestión de bases de datos que soporta MySQL y PostgreSQL simultáneamente. Incluye procedimientos almacenados, funciones y consultas optimizadas para alto rendimiento.

### ✅ Características

- Soporte multi-motor: MySQL y PostgreSQL
- Procedimientos almacenados y funciones
- Consultas optimizadas con índices
- Migraciones y control de versiones del esquema
- Dashboard de monitoreo de rendimiento

### 🛠 Stack

| Capa | Tecnología |
|------|-----------|
| Motor 1 | MySQL 8 |
| Motor 2 | PostgreSQL 15 |
| Backend | .NET Core |
| ORM | Dapper + EF Core |

## 🧪 `04` — Suite de Testing

> **Colección completa de pruebas para APIs .NET Core**

![xUnit](https://img.shields.io/badge/xUnit-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)
![Postman](https://img.shields.io/badge/Postman-FF6C37?style=for-the-badge&logo=postman&logoColor=white)
![Swagger](https://img.shields.io/badge/Swagger-85EA2D?style=for-the-badge&logo=swagger&logoColor=black)

### ¿Qué hace?

Suite completa de testing que combina pruebas unitarias con xUnit, colecciones de Postman para pruebas de integración y documentación interactiva con Swagger para validación manual.

### ✅ Características

- Pruebas unitarias con xUnit y Moq
- Colecciones Postman con scripts de validación
- Testing interactivo via Swagger UI
- Cobertura de código con Coverlet
- CI/CD con GitHub Actions

### 🛠 Stack

| Herramienta | Uso |
|-------------|-----|
| xUnit | Pruebas unitarias |
| Moq | Mocking de dependencias |
| Postman | Pruebas de integración |
| Swagger | Testing interactivo |
| Coverlet | Cobertura de código |
